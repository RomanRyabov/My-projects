# Структура работы

1. Подготовить данные к анализу и убедиться, что соблюдены принципы A/B-тестирования (в каждой группе собрано достаточно данных и длительность теста одинакова, группы сбалансированы и отсутствуют пересечения в группах);
2. Визуально оценить интересующие метрики в каждой из групп, чтобы сделать первичные выводы;
3. Убедиться в том, что метрики стабилизированы и можно проводить статистические тесты (помним, о проблеме подглядывания). Для этого чаще всего используются кумулятивные метрики;
4. Если метрика стабилизировалась, то можно проводить статистические тесты, которые позволяют проверить гипотезу о равенстве показателей в группах при альтернативных гипотезах, но предварительно необходимо узнать распределение показателя, который пытаемся сравнить;
5. В комплексе со статистическими тестами используем доверительные интервалы для параметра, чтобы оценить его возможные границы;
6. На основе результатов статистических тестов и построенных доверительных интервалов формируем вывод и принимаем решение.
